🆘 TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Problem: Prince not found
  → Install from https://www.princexml.com/download/
  → Verify: prince --version

  Problem: PDF not generating
  → Check Prince is installed
  → Verify HTML/image file paths
  → Review build output for errors

  Problem: Text/images cut off
  → Reduce font size or padding
  → Increase page margins
  → Use page-break-inside: avoid;

  Problem: Colors look wrong
  → Check CSS is linked correctly
  → Verify hex codes (#RRGGBB)
  → Test in different PDF viewer
