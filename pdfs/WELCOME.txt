🎯 PROFESSIONAL DESIGN TIPS
════════════════════════════════════════════════════════════════════════════════

For $100k-Quality PDFs:

✓ CONSISTENCY
  • Use same color palette throughout
  • Keep 2 fonts maximum (1 for headings, 1 for body)
  • Maintain consistent spacing and alignment
  • Use grid-based layouts

✓ WHITE SPACE
  • Don't cram text - let it breathe
  • Minimum 1.5em between sections
  • Margins of at least 2cm on all sides
  • Use light backgrounds sparingly

✓ TYPOGRAPHY
  • Use bold for emphasis, not size changes
  • Proper hierarchy: h1 > h2 > h3 > p
  • Line-height of 1.8 for body text (readability)
  • Professional font pairing (sans-serif + serif)

✓ IMAGES
  • High resolution (300+ dpi for print)
  • Professional photography
  • Consistent sizing and placement
  • Relevant to the content

✓ BRANDING
  • Logo visible and prominent
  • Brand colors used consistently
  • Brand voice maintained throughout
  • Clear contact information

✓ LAYOUT
  • Clear visual hierarchy
  • Logical page breaks
  • Centered and aligned elements
  • Grid-based design system

🆘 NEED HELP?
════════════════════════════════════════════════════════════════════════════════

Prince not found?
  → Install from https://www.princexml.com/download/
  → Verify: prince --version in terminal

PDF not generating?
  → Check Prince is installed
  → Verify HTML and CSS file paths are correct
  → Review build output for error messages

Colors look wrong?
  → PDF colors may render slightly different than screen
  → Check hex code format (#RRGGBB)
  → Test in different PDF viewer

Text cut off?
  → Reduce padding or font size
  → Increase page margins: @page { margin: 3cm; }
  → Use page-break-inside: avoid; on large elements

For more help:
  → See SETUP-GUIDE.md troubleshooting section
  → See QUICK-REFERENCE.txt for common fixes
